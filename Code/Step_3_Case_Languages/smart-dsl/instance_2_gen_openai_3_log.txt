## Differences: Grammar 2 vs Grammar 1

### 1) Renaming: **Modifier → Validator**
- **Grammar 1:** `Modifier` rule introduced with keyword `'Modifier'` (capital M).

- G1: `Modifier onlyOwner (...) { ... }`
- G2: `validator onlyOwner (...) { ... }`

So:
- `modifiers+=Modifier` was removed
- `validators+=Validator` was added (replacement)
- `dataStores+=DataStore` was added (new feature)
